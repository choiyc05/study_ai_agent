 ## 온톨리지(Ontology)

### 1. 철학적 의미: 존재론
```bash
'존재(Onto)' + '학문(Logia)' 합성어
```

- **본질 탐구**: 세상에 존재하는 사물이나 개념의 본질이 무엇인지, 그리고 그것들이 어떤 분류에 속하는지를 연구
- **보편적 범주**: 실재하는 것, 공통적인 속성 => 세상을 이해하는 기본적인 틀(Framework) 제공

---
### 2. 주요 특징 및 역할
```bash
'지식 공유' + '재사용' 정형화
```
 
- 주요 구성 요소
    
        클래스(Class): 개념의 집합 (예: '과일', '자동차')
        인스턴스(Instance): 구체적인 개체 (예: 과일 중 '사과', 자동차 중 '내 차')
        속성(Property): 개체가 가진 특징 (예: 사과의 색상은 '빨강')
        관계(Relation): 개념 간의 연결 고리 (예: 사과는 과일의 '일종이다')

```bash
개체(individuals/instance)
가장 기초적인 바닥 단계(Ground level)의 객체다.
구체적인 사물(사람, 테이블)뿐만 아니라 추상적인 것(숫자, 단어)도 포함될 수 있다.
클래스(class)
개체들의 집합, 개념, 또는 유형(Type)이다.
개체가 충족해야 하는 조건(속성)을 정의한다.
속성(attributes)
개체나 클래스가 가지는 특징, 특성, 매개변수다.
이름, 부품(part), 숫자 데이터 등이 포함된다.
관계(relations)
개체와 개체, 혹은 클래스와 클래스가 서로 어떻게 연결되는지 규정한다.
상하 관계(subsumption): is-a-subclass-of (분류 체계 형성)
부분 관계(mereology): part-of (복합 객체 형성)
도메인 특화 관계: made-in 같은 특정 분야의 사실 관계
위의 기본 구성 요소에 논리(logic) 및 제약 구성 요소(constraints)에 관련된 구성 요소들이 있다.

함수 용어(function terms)
문장에서 개체 대신 사용할 수 있는 복잡한 구조다.
제약 조건(restrictions)
어떤 주장이 참이 되기 위해 반드시 만족해야 하는 조건이다.(입력 값 검증 등)
규칙(rules)
“만약 A라면 B이다(If-Then)” 형태의 문장으로, 논리적 추론을 가능하게 한다.
공리(axioms)
도메인 이론 전체를 구성하는 논리적 주장(assertions)이다.
여기에는 기본적인 참인 명제뿐만 아니라 그로부터 유도된 이론들도 포함된다.
```
##### [wikipedia](https://en.wikipedia.org/wiki/Ontology_components)


- 주요 역할

        지식의 구조화: 방대한 데이터를 컴퓨터가 이해할 수 있는 계층 구조로 만듭니다.
        공유 및 재사용: 서로 다른 시스템 간에 동일한 용어와 개념을 사용하여 데이터 호환성을 높입니다.
        추론(Reasoning): 정의된 관계를 바탕으로 직접 명시되지 않은 새로운 사실을 찾아냅니다. 
        (예: 'A는 B의 자식이다'와 'B는 C의 자식이다'를 통해 'A는 C의 손자이다'를 도출)


### 3. AI에서 필요한 이유
```bash
단순한 데이터 나열이 아닌 '이해', '맥락' 등을 만들기 때문
```

- 시맨틱 웹(Semantic Web) 구현: AI가 단어의 단순한 철자가 아니라 그 **의미(Context)**를 파악. 문맥적인 이해에 도움. '생성형 AI' / 'LLM' + 'Ontology'

- 할루시네이션(환각 현상) 방지: LLM은 확률 기반 예측이므로 사실이 아닌 내용을 말할 수 있다. 이때 온톨로지 기반의 **지식 그래프(Knowledge Graph)**를 활용하면 검증된 사실에 기반한 답변을 생성이 가능.

- 데이터 통합: 기업이나 연구소의 파편화된 데이터를 하나의 체계적인 지식 베이스로 통합하여 AI가 학습하거나 참조하기 좋은 형태로 만들어 줍니다.

- 복잡한 문제 해결: 단순 답변을 넘어, 개념 간의 복잡한 인과관계를 분석해야 하는 의료, 법률, 금융 분야의 전문 AI 서비스에서 필수적인 뼈대 역할을 합니다.
